About Property based testing
Property based testing checks certain propertied hold true for all possible input. It does this by automatically generating random sample of valid inputs.
More about property based testing

Clojure spec is for

  • Documentation
  • Validation
  • Error reporting
  • Destructuring
  • Instrumentation
  • Test-data generation
  • Generative test generation
More about Clojure spec
All of the spec features are for clojure but it is also very easy to integrate Test-data generation and Generative test generation features to any type (new or legacy) of java application.
